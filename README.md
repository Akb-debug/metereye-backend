# MeterEye AI – Backend

MeterEye AI is a smart meter monitoring system that combines IoT, OCR, and modern backend architecture to automate electricity consumption tracking.

## 🚀 Features
- User authentication (JWT)
- Meter management (Classique & CashPower)
- Multi-source readings:
  - Manual input
  - ESP32-CAM (image upload)
  - Sensor-based readings
- Device (ESP32) onboarding & monitoring
- Image processing pipeline (OCR-ready)
- Consumption analytics & statistics

## 🛠️ Tech Stack
- Java 21
- Spring Boot
- Spring Security (JWT)
- JPA / Hibernate
- PostgreSQL
- REST API

## 📡 IoT Integration
Supports ESP32-CAM modules:
- Device registration & association
- Heartbeat monitoring
- Image upload for OCR processing
- Offline buffering support
